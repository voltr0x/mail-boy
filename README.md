# Mail Boy

A learning project built to explore the **Model Context Protocol (MCP)** architecture and autonomous AI agents using LangGraph.

## Tech Stack
- **Backend:** Node.js, Express, MCP SDK, ImapFlow (Gmail X-GM-RAW support).
- **Frontend:** React (Vite), TailwindCSS, LangGraph JS, Gemini 2.0 Flash.

## How to Run

### 1. Backend (MCP Server)
```bash
cd server
npm install
npm run dev
```
Running on: `http://localhost:3001`

### 2. Frontend (Client)
```bash
cd client
npm install
npm run dev
```
Running on: `http://localhost:5173`

## Usage
1. Open the web app.
2. Click the **Settings (Gear Icon)**.
3. Enter your **Gemini API Key** and **IMAP Credentials** (Use a Gmail App Password).
